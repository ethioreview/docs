---
title: Telegram setup checklist
description: Step-by-step dashboard workflow for Telegram integration.
---

<Steps>
  <Step title="Add bot to group">
    Search for the EthioReview bot (see dashboard for `@username`). Add it to your customer or internal feedback group.
  </Step>
  <Step title="Generate link code">
    Dashboard → **Integrations → Telegram → Generate link code**
  </Step>
  <Step title="Connect">
    In the group: `/connect YOUR_CODE` or use the **Add to group** link.
  </Step>
  <Step title="Verify">
    Dashboard status shows **CONNECTED** with group name.
  </Step>
  <Step title="Test collection">
    As group admin, run `/collect`. Confirm review link opens correctly.
  </Step>
  <Step title="Enable notifications">
    Toggle notifications ON to receive alerts for new published reviews.
  </Step>
  <Step title="Train staff">
    Document `/collect` for team members who post review requests after service.
  </Step>
</Steps>

## Troubleshooting

| Problem | Fix |
| ------- | --- |
| `/connect` fails | Code expired — generate a new one |
| `/collect` denied | User must be Telegram group admin |
| No notifications | Confirm notifications are enabled and the group is **CONNECTED** |
| Wrong org linked | Unlink and reconnect with fresh code |
