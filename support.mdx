## Contact

- Email: `developers@ethioreview.com` (replace with your team address)

## Reporting bugs

Include:

1. Organization profile ID (not secret keys)
2. Environment (sandbox vs production)
3. Request ID / correlation header if available
4. Expected vs actual behavior

Never share `sk_*` secrets in support tickets.
